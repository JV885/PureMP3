# JV PureMP3: The Ultimate Audio Pipeline

![APP LOGO](https://raw.githubusercontent.com/JV885/PureMP3/main/asset/icon.png)

JV PureMP3 is a high-performance, precision-engineered YouTube-to-MP3 extraction tool. Designed for audiophiles and car enthusiasts, it focuses on format compatibility, clean metadata, and a premium user experience.

---

## Key Features

Precision Audio Extraction
- High-Fidelity MP3: Extracts audio at the highest available bitrate (192kbps/320kbps).
- Sequential Formatting: Automated prefixing (e.g., 001 - Title.mp3) for perfect playback order on legacy car stereos and MP3 players.
- Smart Metadata Scrubbing: Removes "Official Video", "Lyrics", and other junk from titles automatically.

Advanced Interactive Dashboard
- Real-Time Analytics: Clickable counters for Done, Failed, and Skipped tracks.
- Detailed Batch Overviews: Inspect exactly which tracks were processed in each session.
- Smart History: Automatically skips already-downloaded tracks to save time and bandwidth.

Integrated Smart Player
- Local Library Browser: Search and play your entire collection within the app.
- ID3 Tagging: Automatic injection of clean Title and Artist metadata.
- Intuitive Controls: Shuffle, Repeat, Seek, and Volume control with modern aesthetics.

Privacy & Performance
- Zero Tracking: 100% local processing. Your data stays on your machine.
- Multi-Threaded Engine: Simultaneous search and queuing for rapid batch processing.
- Auto-Update Ready: Seamlessly checks for new versions via GitHub.

## Installation & Setup

For Users
1. Download the latest jv_puremp3.exe from the [Releases](https://github.com/JV885/PureMP3/releases) page.
2. Launch and enjoy! No installation required (Portable).

For Developers
1. Clone the repository: git clone https://github.com/JV885/PureMP3.git
2. Install dependencies: pip install -r requirements.txt
3. Ensure ffmpeg.exe and ffprobe.exe are in the root directory.
4. Run: python jv-puremp3.py
